Cloud Migration Project

Project Overview

This project demonstrates my cloud engineering skills by migrating an on-premises infrastructure to AWS. It leverages Terraform for infrastructure as code and Ansible for automation.

The project showcases:

  • Provisioning EC2 instances and S3 buckets
  • Infrastructure automation with Terraform
  • Configuration management and deployment with Ansible
  • Version control using Git and GitHub

How to Run

To try the project yourself:

  • Clone the repo: git clone https://github.com/edebo98/cloud-migration-project.git
  • Run terraform init in the Terraform folder
  • Run terraform apply to provision resources
  • Use Ansible playbooks to configure the instances
